# JET Cloud Inverter Integration

A custom component for Home Assistant that connects to the JET Cloud API to monitor your microinverters.

Features:
* Config Flow support (setup via UI).
* Real-time grid and PV power monitoring.
* Native Home Assistant Energy Dashboard compatibility.
* Automatic API token refresh handling.

**Important**: Maintain the default 60-second polling interval to avoid API rate limiting.
